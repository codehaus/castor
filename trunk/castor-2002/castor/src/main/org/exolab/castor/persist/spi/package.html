Defines the interfaces supported by a Castor persistence engine service provider. A persistence engine server provide implements the {@link org.exolab.castor.persist.spi.PersistenceFactory} factory to produce persistence engine for a given object descriptor. Each persistence engine implements the {@link org.exolab.castor.persist.spi.Persistence} interface against the persistent storage and the {@link org.exolab.castor.persist.spi.PersistenceQuery} interface to perform queries.
