Use {@link org.exolab.castor.jdo.JDO} to open a database, open a transaction and load database configuration and mapping files.

The following example loads JDO configuration file specifying Java/SQL mapping, opens up a database for read/write access, and performs a query: 

  JDO         jdo;
  Database    db;
  Transaction tx;
  OQLQuery    oql;

  // Define a new database source
  jdo = new JDO();
  jdo.setDatabaseName( "test" );
  jdo.setConfiguration( "database.xml" );

  // Open a new database, begin a transaction
  db = jdo.getDatabase();
  db.begin();

  // Create and execute a query
  oql = db.getOQLQuery( "SELECT p FROM Product p" );
  oql.execute();

  . . .

  // Commit transaction, close database
  db.commit();
  db.close();
