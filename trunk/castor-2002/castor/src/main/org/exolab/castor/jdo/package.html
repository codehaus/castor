The ODGM interface to the JDO engine.

Version:
$Revision$ $Date$
Author:
Assaf Arkin

Use {@link org.exolab.castor.jdo.ODMG} to obtain an implementation of the JDO engine and load mapping files. The ODMG API can be used to create a new database, open a transaction, etc.

The following example loads JDO configuration file specifying Java/SQL mapping, opens up a database for read/write access, and performs a query: 

  ODMG        odmg;
  Database    db;
  Transaction tx;
  OQLQuery    oql;
 
  odmg = new ODMG();
  odmg.loadDatabase( "database.xml", null );

  db = odmg.newDatabase();
  db.open( "test", db.OPEN_READ_WRITE );
  tx = odmg.newTransaction();
  tx.begin();

  oql = ODMG.newOQLQuery();
  oql.create( "SELECT p FROM Product p" );
  oql.execute();
